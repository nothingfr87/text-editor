# Text Editor
A Simple text editor made in Python with `tkinter` and `filedialog`

---
## Contents:


--- 
## Requirements
- Python 3.7+:
    - CustomTkinter
    - filedialog (built-in tkinter)

    ```bash
    # Make a Virtual Environment

    # Linux/Mac
    python3 -m venv .venv

    # Windows
    py -m venv .venv

    # Activate the Virtual Environment

    # Linux/Mac
    source .venv/bin/activate

    # Windows
    .venv\Scripts\activate.bat

    # Download required packages
    pip install customtkinter
    ```

## Installtion:
You'll find an executable in [Releases Tab](https://github.com/peanutbutte-r/text-editor/releases), run it and you can use the text editor
